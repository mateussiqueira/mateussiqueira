Um repositório completo com 14 desafios técnicos simulando problemas reais do mercado financeiro brasileiro.

**Destaques:**

- ✅ NFS-e Integration (SOAP/XML)

### ⚡ **Nidus** — Self-hosted Deploy Platform (Vercel-like)
Plataforma de deploy que roda em sua máquina. Conecte GitHub, faça push, e o sistema constrói + executa automaticamente.

**Destaques:**
- 🐳 Deploy isolado em Docker containers
- 🔄 Webhook automático do GitHub
- 🔐 Segurança de aplicações
- ⚙️ Worker em Go (15MB vs 100MB+ em Node)
- 📦 Suporte a múltiplas linguagens

**Stack:** Next.js 16, NestJS, Prisma, PostgreSQL, Redis, Caddy, Docker, Go, CLI

**Repo:** https://github.com/mateussiqueira/nidus

### 🔐 **Unleash** — MDM Bypass para macOS (Recovery)
Script shell single que faz bypass completo de MDM em Apple Silicon e Intel.

**Camadas de proteção:**
1. Remoção de DEP activation records
2. Bloqueio de rede (/etc/hosts + IPv6)
3. Desabilitação de daemons launchd
4. Limpeza de artefatos por usuário
5. pf Firewall (kernel-level)

**Stack:** Shell Script, macOS recovery, pf firewall

**Repo:** https://github.com/mateussiqueira/unleash
